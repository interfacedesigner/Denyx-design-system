## 컴포넌트 계층 — Primitives → Composite → Shell

<div style={{ display: 'flex', alignItems: 'center', gap: 8, margin: '16px 0', flexWrap: 'wrap' }}>
  <div style={{ padding: '10px 20px', borderRadius: 6, background: '#E3F2FD', fontWeight: 700, fontSize: 13 }}>Primitives</div>
  <span style={{ fontSize: 18, color: '#999' }}>→</span>
  <div style={{ padding: '10px 20px', borderRadius: 6, background: '#FFF3E0', fontWeight: 700, fontSize: 13 }}>Composite</div>
  <span style={{ fontSize: 18, color: '#999' }}>→</span>
  <div style={{ padding: '10px 20px', borderRadius: 6, background: '#F3E5F5', fontWeight: 700, fontSize: 13 }}>Shell</div>
</div>

| 계층 | 정의 | 독립 생성 | 예 |
|---|---|---|---|
| **Primitives** | 혼자서 렌더. DS import 없음 | **가능** (유일) | Button, Modal, AiCard, DataTable |
| **Composite** | Primitives를 1단계 조합 | 불가 | PageHeader, Sidebar, FilterBar |
| **Shell** | 최종 페이지 골격 | 불가 | DashboardLayout, DenyxAiWidget |

```
Tokens  ←  Primitives  ←  Composite  ←  Shell
```

## AI 대화형 개발

AI 에이전트(Claude, Cursor 등)를 MCP 서버에 연결하면 **프롬프트로 UI를 조립**할 수 있습니다.

### MCP 서버 연결

```json
{
  "mcpServers": {
    "denyx-ds": {
      "url": "https://denyx-design-system-storybook.vercel.app/api/mcp"
    }
  }
}
```

### 사용 가능한 도구

| 도구 | 설명 |
|---|---|
| `list-components` | 전체 컴포넌트 목록 (계층별 필터) |
| `get-component` | 컴포넌트 상세 (props, 사용법, 예제) |
| `get-tokens` | 디자인 토큰 정의 (색상, 타이포, 간격) |
| `get-patterns` | 합성 패턴 + Do/Don't 규칙 |

### 프롬프트 예시

```
"Denyx DS로 DB 인스턴스 목록 페이지를 만들어줘.
 DashboardLayout + PageHeaderAiInline + DataTable 조합.
 필터는 FilterBar에 severity 드롭다운."
```

```
"현재 페이지 헤더를 PageHeader에서 PageHeaderAiInline으로 교체해줘.
 AI 프롬프트 suggestion은 '사용량 분석', '병목 조회' 2개."
```

### 컨텍스트 주입 (MCP 없이)

```bash
curl https://denyx-design-system-storybook.vercel.app/design-system/DESIGN.md > denyx-ds.md
```

이 파일을 AI에게 전달하면 전체 카탈로그를 인지한 상태로 코딩합니다.

## 5 핵심 규칙

1. **색상은 토큰만** — 인라인 hex 금지. `tokens.css`의 `:root` 변수 사용.
2. **AI 카드는 `<AiCard>`** — 위젯 안 모든 카드는 이 컴포넌트로 감싸기.
3. **헤더는 절대 숨기지 않음** — AI 위젯이 열려도 PageHeader의 모든 요소 유지.
4. **SVG 텍스트는 단위 필수** — `fontSize="9px"` (단위 없는 숫자 금지).
5. **DataTable density 기본 `default`** — compact는 100+ 행 한정.

## 빠른 시작

### 신규 페이지

```tsx
import {
  DashboardLayout, PageHeaderAiInline,
  DenyxAiWidget, useDenyxAi,
  type AiPromptSuggestion,
} from "@denyx/design-system";

const SUGGESTIONS: AiPromptSuggestion[] = [
  { label: "DB 사용량 분석", query: "Oracle DB 24시간 사용량" },
  { label: "병목 조회",     query: "느린 쿼리 TOP 5" },
];

export default function MyPage() {
  const { aiActive, setAiActive } = useDenyxAi();
  return (
    <DashboardLayout
      activeProduct="db"
      widgetOpen={aiActive}
      header={<PageHeaderAiInline
        title="DB / oracle_dnx"
        promptSuggestions={SUGGESTIONS}
        onPromptSubmit={(q) => { setAiActive(true); }}
      />}
      main={<MyContent />}
    >
      <DenyxAiWidget open={aiActive} onClose={() => setAiActive(false)} />
    </DashboardLayout>
  );
}
```

### 신규 AI 응답 메시지

```tsx
import {
  AiCard, AiSectionHeading, AiBulletList,
  type SemanticIntent,
} from "@denyx/design-system/widget";

function CriticalFinding({ intent, title, bullets }: {
  intent: SemanticIntent; title: string; bullets: string[];
}) {
  return (
    <AiCard delay={0}>
      <AiSectionHeading tone={intent}>{title}</AiSectionHeading>
      <AiBulletList items={bullets} />
    </AiCard>
  );
}
```
